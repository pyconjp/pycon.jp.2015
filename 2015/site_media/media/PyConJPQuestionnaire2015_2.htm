Q11 講演について、感想、要望があれば教えて下さい / Let us know your feedback or recommendations regarding the presentations.

talkで立ち見が多かったため、会場のキャパシティについてはある程度事前調査をして、適切な配置になるように最適化できる余地があるように感じました。
直近で実用的なトークと興味があるのトークだと、どうしても実用的なトークを選んでしまうのが悩ましいですね。
会場を１つ増やして，もう１セッション並行で出来るようにして選択肢を増やして欲しい。
日本語公演の質疑応答時間をもう少し多めにとって欲しかった。
活用事例が多かったが、技術的内容のセッションが薄い気がする
keynoteで皆に通訳機を配るのはすごいと思います。
今回聴けた講演についてはどれも素晴らしく、勉強になるものもかなり多かったです。ありがとうございました。
カンファレンスホール以外は基本的に部屋が狭く、立ち見が出ていたのは不満でした。
よく準備された講演ばかりでとても興味深かったです。
Conference Hall が少し暑かった
カンファレンスルーム１で参加した講演が、どれも超満員となっていたので、今後はもう少し広い部屋で開催頂けると幸いです。
席が少なくて立ち見が多かった印象でした。 もう少し席の数が多いと良かったかなという要望です。
ベストトーク賞的なものがあるのはどうでしょうか
講演が、「非常に簡単なもの」と「狭い分野の深い話と」両極であつた。 も少し、概論的な講演を望む。
時間管理と、それによる質問時間の確保
満員で入れない講演があったのが残念だった。
It is great to be able to learn alot on the server and web service and system framework through Python. It would be great to cover other areas that use Python alot. For example, in 3D Animation, games and Vfx industry, there are many usage of python that would be interesting to show to people from different industry that uses Python.
カンファレンスルーム１の椅子がスグになくなってしまったのが（特に初日）少し残念でした。中継を映すサテライトルームとかがあっても良いかもです。
言語自体やフレームワークなど、開発に関連する技術をより 深く知るようなセッションがもっと多くあると良いなと思いました。 たとえばパッケージングについてのセッションで、各ツールの最近の動向についての話が中心ですが、そもそもそれらをどう使い分けるのかは既に認識している 事が前提になっており、もっとパッケージング自体の概要レべルから知りたち思っている私にとっては情報としてあまり有用なものでありませんでした。
充実していたが、たくさんあっても技術よりの話は疲れるので、１日で十分だと感じた。
カンファレンスホール以外の2会場がいつも満席だった印象を持ちました。 今回の会場は全体的にコンパクトにまとまっていて好きなのですが、こちらよりも広い会場も検討いただけると幸いです。
会議室1は席数が少なく、つらかったです。 補助いすで各自が配置できればよいのかな？と少し思いました。 あと、テーブルを完全に除けば、いすだけならもっと配置できるのでは、とも思いました。
事例よりも、pandas internalのようなもう少しコアな技術よりの話の割合が多いと嬉しいです
Web関連のセッションをもう少しお聞きしたかったです。
5トラックくらいにして、深いニッチな話も聞きたい
データサイエンス・機械学習系以外のテーマをもう少し聞きたい。
CythonやKaaなどのトークが採用されていたら聞きたかったのですが、今回は割とプログラミング技術的に深いトークが少なかったように思うので、そこだけが少し残念でした。
実装よりの発表(コードのある発表)を増やして欲しい
（私的理由により参加できなかったという理由で恐縮であるが、）Youtubeに可能な限り動画をアップロードしていただきたい。
Keynoteに関しては、非常に興味深い内容でした。 LTなども、ところどころ面白いので、興味を持って聞くことができました。
"Possibilities of Python"という割には多様性に欠けていたように思えます。結局おなじみのPythonの応用事例について報告があるだけで、新しい知見は特にありませんでした。
LTはちょうどよい人数だった。
書かれていた対象リスナーと実際のリスナーが乖離していた印象がある。
